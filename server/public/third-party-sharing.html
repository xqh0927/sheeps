第三方信息共享清单

本清单旨在向您披露「金色体育」App 向第三方共享个人信息的情况。我们承诺：仅在必要且合法的前提下进行共享，并严格要求第三方按照本清单约定使用相关信息。

一、共享原则

  • 合法正当：仅在法律法规允许、取得您明确授权或履行法定义务的情形下共享
  • 必要最小：仅共享实现业务目的所必需的最少信息
  • 安全保护：通过协议约束、技术监控等方式确保第三方安全使用
  • 透明公开：通过本清单向您完整披露共享情况

三、我们的安全保障措施

  • 与所有第三方签署严格的数据处理协议（DPA），明确数据使用范围与安全责任
  • 定期审计第三方的数据安全合规情况
  • 采用 API 签名、HTTPS 加密等手段防止数据在传输过程中被窃取
  • 如第三方违反约定，我们会立即终止合作并要求删除数据

四、如何行使您的权利

  • 如您不同意某项共享行为，可在系统"设置"中关闭相关功能
  • 您可以随时撤回对第三方的授权（可能影响部分功能正常使用）
  • 如有任何疑问或投诉，可通过 App 内反馈渠道与我们联系
