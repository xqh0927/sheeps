🃏 卡片组件设计

统一的卡片组件库，支持4种卡片样式和游戏专用卡片，提升视觉层次感。

标准卡片 (Elevated)
带有阴影效果的标准卡片，适用于大部分内容展示场景。悬停时有上浮动画效果。
描边卡片 (Outlined)
带有金色描边的卡片，适用于需要强调边界的内容。风格典雅，符合国风设计。
渐变卡片 (Gradient)
带有朱砂红到帝王金渐变的卡片，适用于特别推荐或重要通知。
🎮 游戏卡片
专为游戏界面设计的卡片，使用朱砂红渐变背景，营造沉浸式体验。
