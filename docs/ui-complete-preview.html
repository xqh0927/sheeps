‹秘境修仙指南
欢迎来到秘境消消乐！请修行者详阅以下修仙法则
1
一、卡牌消除（核心玩法）
点击上方重叠层叠的神秘卡牌，它们会落入下方 7 格的乾坤法宝槽。只要槽内集齐 3 张相同图案的卡牌，即可将其熔炼消除。若 7 个槽位被填满且无法消除，则法决失效，挑战失败。
2
二、仙法宝（三大道具）
移出法宝：将最左侧 3 张卡牌移出法宝槽并暂存，腾出空间。
洗牌符咒：将场上所有未消除的卡牌彻底重新排列，化解死局。
撤销仙法：撤回上一步的熔炼点击，退回上一张卡牌。
3
三、福禄修为（积分规则）
每日签署福禄（签到）可稳定斩获积分。挑战秘境主线关卡，首次破阵通关亦可赢得海量积分。参与多人实时「天命对决」斩获胜绩，更是赢取大量积分与连胜荣耀的最佳途径！
