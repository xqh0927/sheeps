📖 使用指南

设计原则
  • ◆ 国风美学：使用朱砂红、帝王金等传统色彩，标题采用衬线字体
  • ◆ 深色主题：主背景使用深空黑，减少视觉疲劳
  • ◆ 弹性动画：按钮按压、对话框弹出使用弹簧动画
  • ◆ 统一圆角：小圆角 8px，中圆角 12px，大圆角 16px
